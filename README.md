# open-source-survey-2026

This repo contains the results of a community survey, asking about the state of open source in the subsurface and energy domains. These are the main files:

- **`Form.pdf`** — the original form, exported from Google Forms.
- **`Standard_report.pdf`** — the basic report, exported from Google Forms.
- **`Responses.csv`** — the raw data exported from Google Forms to Google Sheets, then to this file. 
- **`Analysis.ipynb`** — a Jupyter Notebook written in Python, that cleans and plots the data.

The folder is a [`uv`](https://docs.astral.sh/uv/) project; clone the repo, install `uv`, and run `uv sync` then `uv run jupyter notebook`.

## Intro text to survey

This survey attempts to capture the activities and attitudes around open source software, open data, and open content in the field of subsurface science and engineering.

The survey will be open from 18 May 2026 until 7 June 2026. The results will be shared at [the _Open for Energy_ workshop](https://eageannual.org/workshop-7/) at the 2026 EAGE Annual Conference in Aberdeen, UK, on 7 June 2026, and published in this GitHub repo.

Thank you for taking part in this anonymous survey!

Matt Hall, @kwinkunks
